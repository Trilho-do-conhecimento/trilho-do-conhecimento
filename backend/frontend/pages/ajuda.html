Perguntas frequentes

Como faço para acessar e baixar meus certificados concluídos?

Você pode visualizar e baixar seus certificados na página “Certificados” após realizar login como aluno.

O projeto prevê assinatura eletrônica?

O projeto prevê a integração futura com a API Adobe Sign para captação de assinaturas digitais.

Quais recursos de acessibilidade o site oferece?

Inclui integração com API VLibras, contraste visual.

Como um aluno acompanha seu histórico de treinamentos?

Na página “Meus Treinamentos”, o aluno pode visualizar histórico, status e emitir o certificado após concluir o curso.
